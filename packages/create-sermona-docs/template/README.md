# Sermona doc site (starter)

Bootstrapped with `create-sermona-docs`. Run `npm install` then `npm run dev`.

- **Sermona design** — `@sermona/vitepress-theme` + `@sermona/tokens`
- **Generator** — from the `sermona-design-system` monorepo:  
  `node path/to/packages/create-sermona-docs/bin.mjs ./my-docs --link path/to/sermona-design-system`

Replace `--link ...` with published versions of `@sermona/tokens` and `@sermona/vitepress-theme` when available on npm.
