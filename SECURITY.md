# Security policy

## Supported versions

This is a **documentation and CSS token** monorepo, not a network service. Security exposure is mainly **supply chain** (dependencies) and **hosted docs** (static site).

We address **high severity** issues affecting the published handbook or installable packages when practical.

## Reporting a vulnerability

Please **do not** open a public issue for undisclosed security problems.

Instead, contact the maintainers through a **private channel** (e.g. GitHub Security Advisories for this repository, or email if your org provides one for the maintainers). Include:

- Description of the issue and impact  
- Steps to reproduce  
- Affected paths or versions if known  

We will acknowledge reasonable reports and coordinate a fix and disclosure timeline.
